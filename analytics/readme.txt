# Analytics - Dashboard de Gastos Pessoais

## Descrição
O **Analytics** é um gerenciador de despesas focado em oferecer uma experiência visual moderna aliada a uma análise de dados precisa. Este mini-projeto foi desenvolvido com o objetivo central de aplicar conceitos avançados de manipulação de **Arrays em JavaScript** e operações da biblioteca **Math**, além de explorar o potencial máximo de animações e transições com **CSS3**.

## Objetivos Técnicos
- **Manipulação de Arrays:** Uso intensivo de `.filter()`, `.reduce()`, `.map()`, `.sort()` e `Set()` para processar transações.
- **Lógica Matemática:** Implementação de estatísticas financeiras como média diária, valores máximos e mínimos.
- **UI/UX Animada:** Interface rica em micro-interações, transições suaves e um sistema de temas (Light/Dark) dinâmico.
- **Internacionalização (i18n):** Suporte nativo para Português (R$) e Inglês ($), adaptando símbolos e formatos de data.

## Tecnologias Utilizadas
- **HTML5:** Estrutura semântica para acessibilidade e SEO.
- **CSS3 (Custom Properties):** Uso de variáveis para fácil manutenção e implementação de Dark Mode.
- **JavaScript (ES6+):** Lógica de persistência no `localStorage`, filtragem dinâmica e integração com APIs de terceiros.
- **Chart.js:** Biblioteca externa para visualização de dados em gráficos de rosca e linha.
- **Google Fonts & FontAwesome:** Tipografia e ícones para refinamento visual.

## Funcionalidades Principais
- **Dashboard Estatístico:** Cards que calculam automaticamente o balanço total, média de gastos e categorias extremas.
- **Filtros Inteligentes:** Filtragem por Dia, Mês e Ano gerada dinamicamente com base nos dados existentes.
- **Sistema de Temas:** Alternância entre Light e Dark mode com persistência de escolha.
- **Gerenciamento de Dados:** Adição e exclusão de transações com atualização em tempo real (Reatividade Manual).
- **Feedback Visual:** Toasts customizados para confirmação de ações e alertas de erro.

---
*Este projeto demonstra a capacidade de transformar dados brutos em insights visuais utilizando tecnologias puras (Vanilla JS).*
